How it works

Runtime model

Local-only overlay

The extension runs in the browser and rewrites the visible Tool Output panel locally. It does not read local databases or upload chat content elsewhere.

Real panel coverage

The overlay mounts inside the actual OpenClaw sidebar, so it follows width changes and panel open/close cycles instead of guessing a floating position.

Verification preserved

The raw output stays available underneath so users can expand it before acting on a summary.

Clean listing posture
